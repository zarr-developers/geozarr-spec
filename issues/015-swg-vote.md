# SWG Vote for Standard Adoption

## Summary

Conduct the formal Standards Working Group vote to adopt GeoZarr as an OGC Standard.

## Labels

- `governance`

## Milestone

v1-vote

## Background

After OAB review and public comment resolution, the SWG conducts a formal vote to adopt the candidate Standard. A successful vote results in GeoZarr becoming an official OGC Standard.

From the GeoZarr charter (Section 4):
> "Reference implementations that fully use GeoZarr should be documented at the same time the candidate Standard goes to vote."

## Process Overview

### 1. Pre-Vote Checklist

Before calling the vote:
- [ ] OAB review complete (#014)
- [ ] All public comments addressed
- [ ] Final candidate Standard document prepared
- [ ] Implementation reports available (minimum 3 implementations for Candidate maturity)
- [ ] SWG members notified of upcoming vote

### 2. Vote Announcement

- [ ] Formal vote announced to SWG members
- [ ] Voting period defined (typically 2 weeks)
- [ ] Final document package distributed
- [ ] Voting instructions provided

### 3. Voting Period

Per OGC policies:
- Only SWG members may vote
- Votes: Yes, No, Abstain
- Quorum requirements must be met
- No votes must include written justification

### 4. Vote Resolution

**If vote passes:**
- [ ] Results documented
- [ ] Standard submitted for final OGC approval
- [ ] Publication process initiated

**If vote fails:**
- [ ] Review "No" vote justifications
- [ ] Determine required changes
- [ ] Return to revision cycle
- [ ] Schedule new vote after changes

### 5. Post-Vote Publication

- [ ] OGC publishes official Standard
- [ ] Press release prepared
- [ ] Community announcement
- [ ] Update convention READMEs to reflect "Stable" status

## Acceptance Criteria

- [ ] Quorum achieved
- [ ] Vote passes per OGC requirements
- [ ] Results documented and submitted to OGC
- [ ] GeoZarr v1.0 published as OGC Standard

## What to Expect

**For SWG Members:**
- Review final document package before voting
- Cast vote during voting period
- Provide justification if voting "No"

**For Community:**
- Vote is limited to SWG members
- Results will be announced publicly
- Standard will be freely available after publication

## Timeline

Typical timeline from vote call to publication:
- Vote announcement: Day 0
- Voting period: Days 1-14
- Results certification: Days 15-21
- OGC publication: Days 22-30+

## Dependencies

- #014 (OAB Review) must be complete
