<title>Unified Model Structure</title>
<p id="_e65124f4-061d-6f01-538f-ef9027d9ff62">This clause defines the structural organisation of datasets conforming to the unified data model (UDM). It consolidates the foundational elements and optional extensions into a coherent architecture suitable for Zarr encoding, while remaining format-agnostic. The model establishes a modular and extensible framework that supports structured representation of multidimensional, geospatially-referenced resources.</p>

<p id="_c6100eca-2d33-f721-69b7-92a5197dbf5c">The model represents datasets as abstract compositions of dimensions, coordinate variables, data variables, and associated metadata. This abstraction ensures that applications and services can reason about the content and semantics of a dataset without reliance on storage layout or specific serialisation.</p>
